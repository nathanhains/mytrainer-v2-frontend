MyTrainer v2 frontend

MyTrainer v2 is an online Social Media platform directed at uploading/ interacting with workouts/ exercises. This is the frontend half of MyTrainer v2 that encompasses the main layout. The backend can be found at https://github.com/nathanhains/mytrainer-v2-backend.

Installation

Run 'npm start' at the command line.

Contributing

Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.

Please make sure to update tests as appropriate.
